## üõ†Ô∏è Technologies Used

### Frontend
* **React.js (Vite)** - For building the user interface.
* **Tailwind CSS** - For modern and responsive styling.
* **React Router Dom** - For client-side navigation.
* **Google OAuth (@react-oauth/google)** - For secure social login.

### Backend
* **Node.js & Express** - Server-side environment and routing.
* **MongoDB & Mongoose** - Database for storing user profiles and chat history.
* **JSON Web Token (JWT)** - For secure session management.
* **CORS** - To handle cross-origin requests between Vercel and Render.

### Deployment & Tools
* **Vercel** - Frontend hosting.
* **Render** - Backend web service hosting.
* **MongoDB Atlas** - Cloud database hosting.
* **Postman** - For API testing and documentation.
